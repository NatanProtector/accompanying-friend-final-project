About the Project

Accompanying Friend is a comprehensive security platform designed to enhance safety and provide peace of mind when navigating potentially dangerous terrain. This innovative project, developed as a final capstone for our software engineering bachelor's degree, addresses real-world security challenges in high-risk areas with cutting-edge technology.

üéØ Project Mission

Our mission is to create a reliable security ecosystem that connects citizens with security personnel in real-time, enabling rapid response to emergencies and providing continuous protection during navigation through potentially dangerous areas.
